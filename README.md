# Телеграм бот игры "Мировое Господство"
* Игра построена на state (состояниях) игрока.
* Игроку предлагается присоединиться к созданной игре по названию МИРА, который отправляется на сервер.
* Далее игрок выбирает для игры одну из предлагаемых МИРом СТРАН. За игроком на сервере фиксируется его МИР, СТРАНА и сохраняется в базе данных.
* Игра основана на 5 раундах, в которых каждая СТРАНа должна отметить одно из действии и отправить на сервер (кн. Отправить).
* Действия, выполняются на деньги:
  * **Развить свои города**
    * Выбрать город
  * **Ядерная программа**
      * Развить (_без развитой яд. программы невозможно создавать ракеты_)
      * Разработать ракету (_ухудшается экология_)
  * **Экология** (развивается для всех и уничтожается при развитии яд. программы)
  * **Бомбить** (_отправить ракету на другую страну_)
    * Выбрать другие СТРАНы
  * **Отправить**
* Чем более развита Ваша СТРАНА, тем больше денег приносит Вам.
